# Fonte Atteron

## Como adicionar a fonte Atteron ao projeto

### Passo 1: Baixar a fonte
Baixe a fonte Atteron de um dos seguintes sites:
- [DaFont](https://www.dafont.com/atteron.font)
- [BefonTS](https://www.befonts.com/atteron-font.html)
- [Font Download](https://font.download/font/atteron)

### Passo 2: Extrair os arquivos
Após baixar, extraia o arquivo ZIP. Você encontrará arquivos com as extensões:
- `.otf` (OpenType Font)
- `.ttf` (TrueType Font)

### Passo 3: Copiar para esta pasta
Copie os arquivos da fonte para esta pasta (`assets/fonts/`):
- `Atteron.otf`
- `Atteron.ttf`

### Passo 4: Verificar
Após copiar os arquivos, recarregue a página no navegador (Ctrl+F5 ou Cmd+Shift+R) para ver a nova fonte aplicada.

## Fontes alternativas (caso não consiga a Atteron)
Se você não conseguir baixar a fonte Atteron, pode usar uma destas alternativas elegantes:
- Playfair Display (Google Fonts)
- Cormorant Garamond (Google Fonts)
- Libre Baskerville (Google Fonts)
